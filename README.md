## 🎯 Objetivo do Projeto

Projeto de **análise exploratória de dados (EDA)** desenvolvido como parte do **Challenge Alura Store**, com o objetivo de extrair insights estratégicos sobre o desempenho de vendas das lojas Alura Store.

A análise busca compreender fatores que impactam o faturamento, as avaliações dos clientes, os custos de frete e as categorias mais lucrativas, com o objetivo de identificar a loja com melhor desempenho e fornecer insights que auxiliem na tomada de decisão estratégica.

- Comparar o **faturamento total entre as lojas**
- Identificar as **categorias mais lucrativas**
- Avaliar a **satisfação dos clientes através das avaliações**
- Analisar o **impacto do frete nas vendas**
- Explorar padrões de vendas com **visualizações de dados**


## 📈 Visualizações Criadas

Durante a análise foram criados diversos gráficos, incluindo:

- 📊 **Faturamento total por loja**
- ⭐ **Avaliação média por loja**
- 🚚 **Média de frete por loja**
- 🏷 **Faturamento por categoria**
- 🔥 **Mapa de calor de vendas**
- 📊 **Faturamento das Top 3 categorias por loja**
